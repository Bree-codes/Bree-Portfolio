# Personal Portfolio Website

A modern, responsive personal portfolio website showcasing skills, projects, and contact information.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Smooth Navigation**: Fixed header with smooth scroll to sections
- **Hero Section**: Eye-catching introduction with call-to-action buttons
- **About Section**: Personal information with skill highlights
- **Projects Showcase**: Featured projects with images and technology tags
- **Skills Display**: Visual skill bars showing proficiency levels
- **Contact Form**: Interactive form for visitors to get in touch
- **Social Links**: Quick access to GitHub, LinkedIn, Twitter profiles

## Sections

1. **Home/Hero**: Main landing with introduction
2. **About**: Personal background and key strengths
3. **Projects**: Portfolio of featured work
4. **Skills**: Technical expertise with progress indicators
5. **Contact**: Contact form and information
6. **Footer**: Social links and copyright

## Design

- **Primary Color**: Blue (#2563eb)
- **Typography**: System fonts for optimal performance
- **Layout**: Clean, modern design with gradient accents
- **Images**: High-quality project screenshots from Unsplash

## Customization

To personalize this portfolio:
- Update name, title, and description in Hero section
- Replace project information with your actual work
- Adjust skill levels to match your expertise
- Add your social media links
- Update profile image
